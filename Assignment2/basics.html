Bootstrap A Mobile-first design

Content

Determine a hiearchy or pecking order for your content. Typically, a mobile site has fewer elements and there is a progressive enhancement of elements as the width of page gets larger.

Bootstrap Grid-System

  • Rows must be contained within .container class
  • Use these rows to create horizontal groups of columns
  • Keep in mind that the width of a container is divided into 12 sections.

HTML coding example of the Bootstrap framework using rows to create columns is shown below:

<div class="container"> <div class= "row"> <div class="col-xs-6"> <h1> 1st column </h1> </div> <div class="col-xs-6"> <h1> 2nd column </h1> </div> </div> </div>
